Welcome to PC Mall webtest Automation script

SVN Version Last Committed On: 10092012
============================================

------------------------------------------------------------------------------------
Prerequisite before you can run the script in any machine:
------------------------------------------------------------------------------------

1. Installed Java (latest version is recommended).
2. Apache ant (latest version is recommended).
3. Canoo Webtest (latest version). See webtest.canoo.com for details


---------------------------------------------------------------------------------------------
How to run the scripts
----------------------------------------------------------------------------------------------
1. Get the latest version in SVN Repository:
https://afs-pw-wsqa05.pcmall.com:8443/svn/ecost/trunk/osrp/staging/pcMall

2. Go to the pcMall directory and execute the batch file: runWebtestPCMall.bat
3. on the command prompt, type the url host to test.
4. Wait the script to finish. Once finished, it will send a report to Manilaqa@pcmall.com
